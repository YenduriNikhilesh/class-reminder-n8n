## Class Reminder Bot (n8n)

This project contains an n8n workflow that sends Telegram class reminders
10 minutes before class time based on a weekly timetable.

## Features
- Runs in IST (Asia/Kolkata)
- Prevents duplicate alerts for 100-minute classes
- Telegram notifications
- Designed for 24/7 cloud hosting (Oracle Cloud Free Tier)
